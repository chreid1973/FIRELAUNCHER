# FireLauncher Download Page

This folder is ready to upload into your hosted `/ppv/` directory.

## Files

- `index.html`
- `firelauncher-test-v1.0.apk`
- `launcher-preview.png`

## Recommended upload target

Upload the contents of this folder to:

- `https://3hpm.ca/ppv/`

That will give you:

- `https://3hpm.ca/ppv/` for the landing page
- `https://3hpm.ca/ppv/firelauncher-test-v1.0.apk` for the APK

## Notes

- This does not interfere with your existing PPV JSON files.
- Your existing `admin.html`, `save.php`, and `events_ppv_*.json` files can stay in the same folder.
- If you build a newer APK later, replace `firelauncher-test-v1.0.apk` and update the version, date, size, and checksum in `index.html`.
